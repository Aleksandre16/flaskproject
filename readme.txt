ფლასკის პროექტი

ეს აპლიკაცია არის მარტივი ვებ აპლიკაცია, რომელიც აშენებულია Flask-ის გამოყენებით.


Მახასიათებლები:
- მომხმარებლის შესვლა და გამოსვლა
- პაროლის ჰეშირება

განაცხადის სტრუქტურა:
- აპლიკაციის მთავარი სკრიპტი არის app.py.

შენიშვნები:
- ამჟამად, ერთეული ტესტები გამორთულია Python 3.10 და ზემოთ თავსებადობის პრობლემების გამო. როდესაც ეს პრობლემები მოგვარდება, ტესტირება flask_testing და unittest2 ხელახლა ჩაირთვება. უნდა გამოვიყენოთ 3.10 ზე ნაკლები ვერსია
